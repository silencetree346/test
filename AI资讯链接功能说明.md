# 🔗 AI资讯链接功能说明

## 📋 功能介绍

当系统使用AI生成备用资讯时，每条资讯的标题都可以点击，跳转到相关新闻的搜索结果页面。

---

## ✨ 功能特点

### 1️⃣ **智能搜索链接**

AI生成的资讯没有真实的原文链接，系统会自动生成百度新闻搜索链接：

- ✅ 提取标题关键词
- ✅ 生成新闻搜索链接
- ✅ 点击可查看相关真实新闻

### 2️⃣ **视觉提示**

- 🔗 标题后显示链接图标
- 鼠标悬停时显示下划线
- 颜色变为紫色，提示可点击

### 3️⃣ **新窗口打开**

- 使用 `target="_blank"` 在新标签页打开
- 不影响当前浏览
- 方便对比查看

---

## 🎯 使用示例

### AI生成的资讯示例：

**标题：**
```
瑞幸咖啡Q4财报超预期，门店数突破18000家 🔗
```

**点击后跳转到：**
```
https://www.baidu.com/s?tn=news&word=瑞幸咖啡+Q4财报+门店
```

**搜索结果包含：**
- 瑞幸咖啡的真实财报新闻
- 门店扩张相关报道
- 行业分析文章

## 🔧 技术实现

### 代码位置

**链接生成函数：**
```javascript
const generateSearchUrl = (title) => {
    const keywords = title.replace(/[，。、！？：；""''（）]/g, ' ');
    return `https://www.baidu.com/s?tn=news&word=${encodeURIComponent(keywords)}`;
};
```

**应用位置：**
1. `getPresetFallbackNews()` - 预设资讯
2. `generateAIFallbackNews()` - AI实时生成

**样式定义：**
```css
.news-title a {
```

## 📞 常见问题

### Q1: 为什么不直接链接到原文？
**A:** AI生成的资讯是模拟的，没有真实原文。使用搜索链接可以帮助用户找到相关的真实新闻。

### Q2: 搜索结果会准确吗？
**A:** 系统提取标题关键词生成搜索链接，通常能找到高度相关的新闻。

### Q3: 能换成其他搜索引擎吗？
**A:** 可以！修改 `generateSearchUrl()` 函数中的URL即可。

### Q4: 真实资讯也是这样的链接吗？
**A:** 不是。真实资讯（36氪、虎嗅等）链接直达原文。
