# 清除缓存方法 🗑️

## ✅ 方法1：使用页面按钮（最简单）

### 步骤：
1. 打开网站：https://silencetree346.github.io/test/
2. 点击页面上的 **"🗑️ 清除缓存"** 按钮
3. 看到提示："✅ 缓存已清除，下次将获取最新数据"

**完成！** 下次点击"获取最新资讯"时会获取最新数据。

---

## ✅ 方法2：浏览器控制台（开发者）

### 步骤：

1. **打开浏览器控制台**
   - Chrome/Edge: 按 `F12` 或 `Ctrl+Shift+I` (Windows) / `Cmd+Option+I` (Mac)
   - Firefox: 按 `F12` 或 `Ctrl+Shift+K` (Windows) / `Cmd+Option+K` (Mac)
   - Safari: 按 `Cmd+Option+C`

2. **切换到 Console（控制台）标签**

3. **输入以下命令并回车：**
   ```javascript
   localStorage.removeItem('coffee_news_cache');
   localStorage.removeItem('coffee_news_cache_time');
   console.log('✅ 缓存已清除');
   ```

4. **刷新页面**，下次获取资讯时会获取最新数据

---

## ✅ 方法3：浏览器开发者工具（可视化）

### Chrome/Edge：

1. 按 `F12` 打开开发者工具
2. 切换到 **Application**（应用程序）标签
3. 左侧找到 **Local Storage**
4. 点击你的网站域名（`https://silencetree346.github.io`）
5. 找到以下两个键并删除：
   - `coffee_news_cache`
   - `coffee_news_cache_time`
6. 右键点击 → **Delete**（删除）

### Firefox：

1. 按 `F12` 打开开发者工具
2. 切换到 **Storage**（存储）标签
3. 左侧找到 **Local Storage**
4. 点击你的网站域名
5. 找到并删除：
   - `coffee_news_cache`
   - `coffee_news_cache_time`

### Safari：

1. 按 `Cmd+Option+C` 打开控制台
2. 切换到 **Storage** 标签
3. 找到 **Local Storage** → 你的网站域名
4. 删除缓存键

## 📊 缓存说明

### 缓存存储的内容：
- **资讯数据**：获取的新闻列表
- **缓存时间**：数据获取的时间戳

### 缓存有效期：
- **5分钟**：5分钟内的数据直接使用缓存
- **超过5分钟**：自动获取最新数据

### 为什么需要清除缓存？
- ✅ 获取最新资讯（忽略5分钟缓存）
- ✅ 解决数据异常问题
- ✅ 测试新功能
- ✅ 强制刷新数据

---

## 🔍 检查缓存状态

### 在浏览器控制台输入：

```javascript
// 检查是否有缓存
const cacheData = localStorage.getItem('coffee_news_cache');
const cacheTime = localStorage.getItem('coffee_news_cache_time');

if (cacheData && cacheTime) {
    const age = Math.round((Date.now() - parseInt(cacheTime)) / 1000 / 60);
    console.log(`📊 缓存存在，已缓存 ${age} 分钟`);
    console.log(`📦 缓存数据:`, JSON.parse(cacheData).length, '条资讯');
} else {
    console.log('📊 无缓存');
}
```

---

## 💡 提示

### 自动清除缓存：
- 缓存超过5分钟会自动失效
- 刷新页面不会清除缓存
- 关闭浏览器不会清除缓存（除非使用隐私模式）

### 清除缓存后：
- ✅ 下次点击"获取最新资讯"会重新获取数据
- ✅ 不会使用旧的缓存数据
- ✅ 获取时间可能稍长（需要重新请求API）

---

## 🚀 快速清除（一键命令）

**复制以下代码到浏览器控制台：**

```javascript
localStorage.removeItem('coffee_news_cache');
localStorage.removeItem('coffee_news_cache_time');
alert('✅ 缓存已清除！');
location.reload();
```

这会清除缓存并自动刷新页面。
