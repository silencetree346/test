# 🎯 预设资讯精简优化说明

## 📋 优化目标

**核心原则**：只保留热门话题，避免搜索结果为旧新闻的冷门关键词。

---

## ✨ 优化内容

### 数量调整

**优化前：** 20条预设资讯  
**优化后：** 12条精选资讯

**减少40%**，只保留最热门、最可能有今日新闻的话题。

---

## ❌ 移除的资讯（8条）

### 1. **概念性话题**（容易搜出旧新闻）

```
❌ 咖啡品牌加速数字化转型，私域流量成关键
原因：数字化转型是长期趋势话题，新闻更新频率低

❌ 植物基咖啡受追捧，健康趋势引领产品创新
原因：健康趋势类话题，缺乏即时性

❌ 咖啡文化体验店兴起，第三空间价值凸显
原因：概念性话题，新闻多为2-3年前的内容
```

### 2. **技术类话题**（新闻周期长）

```
❌ 咖啡供应链数字化升级，区块链技术保障溯源
原因：区块链+咖啡，搜索结果多为2022年旧新闻

❌ 咖啡机器人门店兴起，无人化运营降本增效
原因：机器人咖啡是技术概念，更新频率低
```

### 3. **数据报告类**（更新频率低）

```
❌ 全球咖啡消费量稳步增长，中国市场增速最快
原因：行业报告类，通常季度或年度发布

❌ 新式茶饮市场规模突破1500亿元，增速放缓
原因：市场数据类，年度更新
```

### 4. **市场趋势类**（缺乏即时性）

```
❌ 即饮咖啡市场爆发，便利店成为新战场
原因：市场趋势话题，新闻多为半年前
```

## 💡 话题选择标准

### ✅ 应该保留的话题

1. **高频发生**
   - 新品发布、门店开业
   - 融资消息、价格变动

2. **日常关注**
   - 品牌动态、产品上新
   - 市场竞争、扩张计划

3. **新闻持续**
   - 每周都有相关新闻
   - 用户搜索量大

### ❌ 应该移除的话题

1. **概念性**
   - 数字化转型、第三空间
   - 长期趋势、理论研究

2. **技术性**
   - 区块链、AI、机器人
   - 技术应用、创新概念

3. **数据类**
   - 年度报告、市场规模
   - 增长率、消费数据

4. **更新慢**
   - 政策法规、行业标准
   - 研究报告、白皮书

## 🔄 持续优化机制

### 如何处理新发现的冷门话题

如果在使用中发现某条预设资讯搜索结果为旧新闻：

1. **记录问题**
   - 记下该条资讯标题
   - 检查搜索结果日期

2. **分析原因**
   - 是否为概念性话题？
   - 是否为技术类话题？
   - 是否更新频率低？

3. **立即替换**
   - 移除该条资讯
   - 替换为更热门话题
   - 保持总数为12条

### 话题选择建议

**优先选择：**
- 品牌新品发布
- 融资并购消息
- 门店扩张动态
- 价格市场变动

**避免选择：**
- 技术概念（区块链、AI等）
- 长期趋势（数字化、第三空间等）
- 数据报告（年度、季度数据）
- 理论研究（学术、行业研究）
