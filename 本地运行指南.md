# 本地运行指南 - 咖啡行业智讯日报

## 🎯 快速启动（3个步骤）

### 步骤1：安装 Xcode 命令行工具

**在您的 Mac 终端中**（不是 Cursor，打开"终端"应用）运行：

```bash
xcode-select --install
```

会弹出安装窗口，点击"安装"，等待完成（约5-10分钟）。

---

### 步骤2：启动本地服务器

安装完成后，在终端中运行：

```bash
cd /Users/luckincoffee/test/docs
python3 -m http.server 8000
```

看到类似这样的提示说明成功了：
```
Serving HTTP on :: port 8000 (http://[::]:8000/) ...
```

---

### 步骤3：在浏览器中打开

打开浏览器，访问：

```
http://localhost:8000
```

🎉 现在可以看到实时的咖啡行业资讯了！

## 🔍 检查开发工具是否已安装

在终端运行：
```bash
xcode-select -p
```

如果显示路径（如 `/Library/Developer/CommandLineTools`），说明已安装。
如果显示错误，说明需要安装。

## 📝 常见问题

**Q: 为什么直接双击 index.html 没有数据？**
A: NewsAPI 要求通过 HTTP/HTTPS 访问，直接打开使用 file:// 协议会被浏览器安全策略阻止。

**Q: 端口 8000 被占用怎么办？**
A: 使用其他端口，如：
```bash
python3 -m http.server 8888
```
然后访问 `http://localhost:8888`

**Q: 停止服务器？**
A: 在终端中按 `Ctrl + C`
