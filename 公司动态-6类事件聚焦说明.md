# 公司动态 - 6类重大事件聚焦说明 🎯

## 功能优化

**优化前**：公司动态分类基于公司名称匹配 + 各类事件关键词

**优化后**：**只聚焦6类公司级别重大事件**

---

## 6类重大事件范围

### 1️⃣ 上市/IPO
- 公司上市计划、IPO筹备
- 挂牌交易、主板/创业板/科创板上市
- 港股上市、美股上市
- **关键词**：`IPO`、`上市`、`挂牌`、`listing`、`goes public`

### 2️⃣ 并购/收购
- 公司并购、收购、整合
- 战略收购、资产收购
- 跨境并购
- **关键词**：`并购`、`收购`、`merger`、`acquisition`、`acquires`

### 3️⃣ 战略投资/融资
- A/B/C/D轮融资、Pre-IPO融资
- 战略投资、天使投资、种子轮
- 估值变化、投资方公布
- **关键词**：`融资`、`投资`、`funding`、`valuation`、`series A/B/C`

### 4️⃣ 出海/国际化
- 海外市场扩张计划
- 国际化布局、全球化战略
- 跨境开店、海外门店开业
- **关键词**：`出海`、`海外`、`国际化`、`overseas`、`expansion`

### 5️⃣ 供应链合作
- 供应链战略合作协议
- 供应商签约、长期采购协议
- 供应链整合、战略伙伴关系
- **关键词**：`供应链`、`供应商`、`supply chain`、`partnership`

### 6️⃣ 财报发布
- 季度财报、年度财报
- 业绩公告、营收公布
- 财务数据、利润报告
- **关键词**：`财报`、`业绩`、`earnings`、`revenue`、`quarterly`

## 技术实现

### 1. 分类关键词优化

**修改位置**：`docs/index.html` - `categorizedNews` 计算属性

**核心变化**：
- ❌ **移除**：所有公司名称匹配逻辑（34家公司列表）
- ✅ **保留**：只保留6类事件的关键词（90+个关键词）
- ✅ **提高权重**：事件关键词匹配分数从2分提升到3分

```javascript
// 只检查6类重大事件关键词（不再依赖公司名称）
companyKeywords.forEach(kw => { 
    if(text.includes(kw.toLowerCase()) || originalText.includes(kw)) {
        companyScore += 3; // 提高权重
    }
});
```

### 2. 预设资讯更新

**修改位置**：`getPresetFallbackNews()` 方法

**核心变化**：
- ❌ **移除**：新品发布、门店扩张、产品创新等12条资讯
- ✅ **新增**：15条6类重大事件资讯

**资讯分布**：
- 上市/IPO：2条
- 并购/收购：2条
- 战略投资/融资：3条
- 出海/国际化：3条
- 供应链合作：2条
- 财报发布：2条

### 3. AI 生成提示词优化

**修改位置**：`generateAIFallbackNews()` 方法

**核心变化**：
```javascript
// 优化前
content: `请生成咖啡和奶茶行业的真实资讯...`

// 优化后
content: `请根据你的知识库，生成今日（当天从0点开始）咖啡和奶茶行业的公司重大事件资讯。

**聚焦6类重大事件（仅限这6类）**：
1. 上市/IPO
2. 并购/收购
3. 战略投资/融资
4. 出海/国际化
5. 供应链合作
6. 财报发布`
```
