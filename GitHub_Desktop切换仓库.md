# 🔄 GitHub Desktop 切换当前仓库

## 📋 方法1：使用仓库列表（最简单）

### 步骤：

1. **打开GitHub Desktop**

2. **点击左上角的仓库名称下拉菜单**
   - 在GitHub Desktop窗口的左上角
   - 会显示当前仓库名称（例如：`tongxu`）
   - 点击这个名称

3. **选择要切换的仓库**
   - 会显示所有已添加的仓库列表
   - 点击想要切换的仓库

4. **完成切换**
   - 仓库会立即切换
   - 文件列表会更新为新仓库的内容

---

## 📋 方法2：使用菜单栏

### 步骤：

1. **点击菜单栏 `Repository`**
2. **选择 `Repository Settings...`**
3. **或者直接使用快捷键**：
   - Mac: `Cmd + ,`（打开设置）
   - 然后选择不同的仓库

---

## 📋 方法3：添加新仓库

### 如果要切换到新的本地仓库：

1. **File → Add Local Repository...**
2. **选择新的文件夹**
3. **点击 Add**
4. **新仓库会自动成为当前仓库**

---

## 📋 方法4：从文件浏览器打开

### 步骤：

1. **在Finder中找到项目文件夹**
2. **右键点击文件夹**
3. **选择 "Open with GitHub Desktop"**（如果已安装）
4. **GitHub Desktop会自动打开并切换到该仓库**

---

## 💡 提示

- **当前仓库显示在左上角**
- **点击仓库名称可以快速切换**
- **所有已添加的仓库都会显示在列表中**
