## ⚠️ 常见问题

### Q1: 代理获取失败
如果看到"代理失败"错误：
- **原因**：部分公共代理服务暂时不可用
- **解决**：系统会自动尝试多个代理，稍等片刻
- **影响**：不影响其他来源的资讯获取

### Q2: 获取资讯数量较少
如果资讯数量少于预期：
- **原因**：过去24小时咖啡相关新闻确实较少
- **解决**：扩大时间范围或调整关键词过滤
- **建议**：查看过去7天的资讯

### Q3: 无法连接到某些来源
- **原因**：网络限制或API配额
- **解决**：系统会跳过失败的来源，使用其他可用来源
- **影响**：不影响整体功能

## 🔧 高级配置（可选）

### 配置AI分析（DeepSeek）
1. 注册 DeepSeek 账号获取 API Key
2. 打开 `docs/index.html`
3. 找到 `DEEPSEEK_API_KEY` 配置
4. 填入你的 API Key
5. 保存后刷新页面

### 自定义资讯源
在 `docs/index.html` 中找到 `NEWS_SOURCES` 配置，可以：
- 添加新的RSS源
- 启用/禁用特定来源
- 调整获取数量
