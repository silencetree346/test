========================================
  macOS 一键启动指南
========================================

【方法一：双击 .command 文件（推荐）】

在 Finder 中进入项目文件夹，双击：

1. 「启动服务.command」
   → 自动启动后端服务
   → 首次运行会自动安装依赖
   → 自动检查配置

2. 「打开前端.command」
   → 在浏览器中打开管理界面

就这么简单！

========================================
【方法二：创建桌面快捷方式】

在终端运行一次：

cd /Users/luckincoffee/test
./创建桌面快捷方式.sh

会在桌面创建「☕️ 咖啡调研系统.app」
以后双击桌面图标即可启动！

========================================
【首次使用配置】

第一次双击「启动服务.command」时：
1. 程序会自动创建配置文件
2. 自动打开文本编辑器
3. 在打开的 .env 文件中填入：
   OPENAI_API_KEY=你的密钥
4. 保存文件
5. 再次双击「启动服务.command」

========================================
【文件说明】

启动服务.command
  - 双击启动后端服务
  - 自动检查配置
  - 自动安装依赖
  - 保持终端窗口打开

打开前端.command
  - 双击在浏览器打开界面
  - 自动检查后端是否运行

创建桌面快捷方式.sh
  - 在桌面创建启动图标
  - 只需运行一次

========================================
【使用流程】

启动：
1. 双击「启动服务.command」
2. 等待显示 "Uvicorn running on http://0.0.0.0:8000"
3. 双击「打开前端.command」或直接打开 frontend/index.html

使用：
- 点击「立即生成报告」生成调研报告
- 系统每天上午9点自动生成

停止：
- 在「启动服务.command」窗口按 Ctrl+C
- 或直接关闭终端窗口

========================================
【常见问题】

Q: 双击 .command 文件没反应？
A: 右键点击 → 打开 → 确认打开

Q: 提示"无法打开，因为它来自身份不明的开发者"？
A: 系统偏好设置 → 安全性与隐私 → 点击"仍要打开"

Q: 如何获取 OpenAI API Key？
A: 访问 https://platform.openai.com/api-keys

Q: 终端窗口可以最小化吗？
A: 可以，但不要关闭，服务需要持续运行

Q: 如何停止服务？
A: 在终端窗口按 Ctrl+C 或关闭窗口

Q: 可以设置开机自启动吗？
A: 可以，将桌面的 app 拖到系统偏好设置 → 用户与群组 → 登录项

========================================
【技巧】

1. 将 frontend/index.html 添加到浏览器书签
   方便快速访问

2. 在终端创建别名：
   echo "alias coffee='cd /Users/luckincoffee/test && ./启动服务.command'" >> ~/.zshrc
   
   之后在任何地方输入 coffee 即可启动
