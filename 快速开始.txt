========================================
  咖啡行业调研系统 - 快速开始指南
========================================

【3分钟快速启动】

1️⃣ 安装依赖
   ./install.sh

2️⃣ 配置API密钥
   cd backend
   cp env_template.txt .env
   
   使用文本编辑器打开 .env 文件，修改：
   OPENAI_API_KEY=你的OpenAI密钥

3️⃣ 启动服务
   cd ..
   ./start.sh

4️⃣ 打开界面
   用浏览器打开 frontend/index.html
   或在 VSCode 中右键 index.html → Open with Live Server

========================================
【获取 OpenAI API Key】

1. 访问 https://platform.openai.com
2. 注册/登录账号
3. 进入 API Keys 页面
4. 点击 "Create new secret key"
5. 复制密钥到 .env 文件

注意：需要绑定支付方式，GPT-4 调用按量计费

========================================
【获取 News API Key（可选）】

1. 访问 https://newsapi.org
2. 注册免费账号
3. 获取 API Key
4. 添加到 .env 文件

免费版限制：100次/天
如不配置，系统使用模拟数据

========================================
【邮件推送配置（可选）】

以 Gmail 为例：

1. 开启 Gmail 两步验证
   https://myaccount.google.com/security

2. 生成应用专用密码
   https://myaccount.google.com/apppasswords

3. 在 .env 中配置：
   EMAIL_ENABLED=true
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER=你的邮箱@gmail.com
   SMTP_PASSWORD=16位应用专用密码
   RECIPIENT_EMAILS=收件人@example.com

========================================
【测试API】

启动服务后，可以测试：

cd backend
python test_api.py

将测试所有API端点并显示结果

========================================
【修改定时任务】

默认每天上午9:00生成报告

修改时间：
在 .env 中添加：
DAILY_REPORT_HOUR=14
DAILY_REPORT_MINUTE=30

即可改为每天下午2:30

========================================
【停止服务】

方式1：在终端按 Ctrl+C
方式2：运行 ./stop.sh

========================================
【常见问题】

Q: 端口被占用怎么办？
A: 修改 backend/main.py 最后一行的 port=8000 为其他端口

Q: 无法生成报告？
A: 检查 .env 中 OPENAI_API_KEY 是否正确配置

Q: 新闻数量太少？
A: 配置 NEWS_API_KEY 获取真实新闻数据

Q: 如何查看服务日志？
A: 服务运行时终端会显示所有日志

Q: 报告保存在哪里？
A: 内存中（重启服务会清空），最多保存30份
