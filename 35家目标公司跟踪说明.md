## 跟踪的8类事件

### 战略级事件（6类）
1. **上市/IPO** - 上市计划、IPO筹备
2. **并购/收购** - 公司并购、收购、整合
3. **战略投资/融资** - 融资、投资、估值变化
4. **出海/国际化** - 海外扩张、国际化布局
5. **供应链合作** - 供应链协议、战略合作
6. **财报发布** - 财报、业绩、营收公告

### 营销级事件（2类）
7. **新品发布** - 新品上市、产品创新、联名限定
8. **门店扩张** - 新店开业、门店数增长、区域布局

## 技术实现

### 1. 双重过滤机制

**文件位置**：`docs/index.html` - 第648-676行

```javascript
// 定义35家目标公司列表
const targetCompanies = [
    // 咖啡品牌（21家）
    '星巴克', 'starbucks', '瑞幸', 'luckin', 'manner', 
    '雀巢', 'nestle', 'costa', 'tims', 'seesaw', 
    'm stand', '挪瓦', '三顿半', '永璞', '隅田川', 
    '时萃', 'fisheye', '连咖啡', 'blue bottle', 
    'lavazza', 'dunkin', "peet's", 'illy',
    '库迪', '库迪咖啡', 'cotti', 'cotti coffee', // 🆕
    // 奶茶品牌（14家）
    '喜茶', 'heytea', '奈雪', 'nayuki', '茶百道', 
    '古茗', '蜜雪冰城', 'mixue', '书亦烧仙草', 
    '茶颜悦色', '霸王茶姬', '沪上阿姨', '一点点', 
    '7分甜', 'coco', '益禾堂'
];

// 检查是否包含目标公司名称
const hasTargetCompany = targetCompanies.some(company => 
    originalText.includes(company) || text.includes(company.toLowerCase())
);

// 检查是否包含8类事件关键词
let hasEventKeyword = false;
companyKeywords.forEach(kw => { 
    if(text.includes(kw.toLowerCase()) || originalText.includes(kw)) {
        hasEventKeyword = true;
        companyScore += 3;
    }
});

// 只有同时满足"包含目标公司"和"包含事件关键词"才归为公司动态
if (hasTargetCompany && hasEventKeyword) {
    companyScore += 10; // 额外加分，确保归为公司动态
} else if (!hasTargetCompany && hasEventKeyword) {
    // 如果只有事件关键词但没有目标公司，降低公司分数
    companyScore = 0;
}
```

### 2. 过滤逻辑

| 条件 | 是否包含目标公司 | 是否包含事件关键词 | 结果 |
|------|-----------------|-------------------|------|
| 条件1 | ✅ | ✅ | ✅ 归为"公司动态" |
| 条件2 | ✅ | ❌ | ❌ 不归为"公司动态" |
| 条件3 | ❌ | ✅ | ❌ 不归为"公司动态"（过滤掉其他公司） |
| 条件4 | ❌ | ❌ | ❌ 不归为"公司动态" |

**关键变化**：
- ✅ **新增**：公司名称白名单（35家）
- ✅ **新增**：双重过滤（公司名称 + 事件关键词）
- ✅ **过滤**：其他公司的资讯（即使包含事件关键词）
