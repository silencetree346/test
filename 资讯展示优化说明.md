## 📊 样式设计

### 总结框样式

**视觉特点：**
- 背景色：浅灰色（#f8f9fa）
- 左边框：紫色强调（3px solid #667eea）
- 图标：💡 灯泡（表示洞察）
- 内边距：10px
- 圆角：6px
- 字体大小：0.95em
- 行高：1.6（易读性）

**CSS代码：**
```css
.news-summary {
    color: #4a5568;
    font-size: 0.95em;
    line-height: 1.6;
    margin-top: 8px;
    padding: 10px;
    background: #f8f9fa;
    border-radius: 6px;
    border-left: 3px solid #667eea;
    font-style: normal;
}

.news-summary::before {
    content: '💡 ';
    color: #667eea;
    font-weight: 600;
}
```

## 📊 总结优化

### 写作原则

**好的总结应该：**
1. **客观准确**：基于事实，不带主观判断
2. **信息完整**：包含关键数据和核心观点
3. **长度适中**：60-100字，一句话或2-3句
4. **通俗易懂**：避免专业术语，易于理解
5. **突出重点**：抓住最核心的信息点

**示例对比：**

❌ **差的总结：**
```
星巴克发布了新的财报，业绩不错，门店也在增加。
（信息模糊，缺乏数据）
```

✅ **好的总结：**
```
星巴克中国发布最新财报，第四季度营收同比增长15%，得益于高端产品线和数字化会员体系的双重驱动。同时，门店总数突破7000家，下沉市场扩张成效显著。
（数据明确，原因清晰，趋势明显）
```
